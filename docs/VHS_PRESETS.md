# VHS Presets

Current presets live in:

```txt
apps/api/app/render/presets.py
```

Current presets:

- Classic VHS
- Dirty Camcorder
- Late Night TV
- Retro Tape Glitch

Each preset controls:

- noise
- scanlines
- blur
- contrast
- saturation
- brightness
- sharpen

Later we can add stronger analog effects like color bleed, frame jitter, tape warp, VHS tracking distortion, CRT masks, and per-frame glitches.
